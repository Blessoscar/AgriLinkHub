# AgriLinkHub

A comprehensive agribusiness information website built with Node.js and Express. The site includes pages for services, impact, donations, and contact information.

## Features

- Donation interest form with success notification
- Impact storytelling with qualitative and quantitative metrics
- Informational pages for About, Services, and Contact
- Responsive web interface

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file with your SMTP and session configuration
4. Run the application: `npm start`

## Usage

- Visit the homepage to learn about AgriLinkHub
- Submit donation interest via the Donate page
- View impact stories on the Impact page
- Contact the team directly through the Contact page

## Technologies Used

- Node.js
- Express.js
- EJS
- nodemailer
- express-session
- connect-flash
- dotenv
